# Usage

If you have a Rust build environment and a decrypted IPA file, you can use it as follows:

```bash
cargo install --git https://github.com/yuimarudev/chatgpt-ios-app-ssl-pinning-patcher
chatgpt-ios-app-ssl-pinning-patcher decrypted.ipa
```

If you don't have a build environment, you can also obtain the executable binary from GitHub Actions or the releases page.
